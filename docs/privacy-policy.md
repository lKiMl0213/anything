# Politica de Privacidade

Este jogo esta em fase beta.

## Coleta de dados

A versao atual do jogo nao coleta automaticamente dados pessoais, nao possui login, nao possui anuncios, nao possui compras no aplicativo e nao utiliza analytics.

## Dados salvos localmente

O jogo salva progresso, configuracoes e preferencias localmente no dispositivo. Esses dados sao usados apenas para o funcionamento do jogo.

## Feedback e contato

Se o usuario enviar feedback por e-mail ou outro canal informado no jogo, poderemos receber as informacoes que ele decidir enviar, como descricao do problema, modelo do aparelho, versao do Android, prints, videos ou endereco de e-mail.

## Permissoes

A versao atual busca usar apenas permissoes estritamente necessarias para o funcionamento do jogo. Se novas funcionalidades forem adicionadas no futuro, esta politica sera atualizada.

## Mudancas futuras

Como o jogo esta em beta, funcionalidades podem mudar. Se forem adicionados anuncios, compras, analytics, login, recursos online ou qualquer coleta adicional de dados, esta politica sera atualizada.
