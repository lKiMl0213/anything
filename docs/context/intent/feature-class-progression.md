# Feature: Class Progression

## What
Class progression lets the player start from a base class, unlock a second class, then unlock a specialization. It also allows talent investment to shape a build over time.

## Why
This feature gives the player long-term identity and meaningful build choice. It supports replayability by making progression paths feel distinct rather than interchangeable.

## Acceptance Criteria
- [ ] The player starts from one base class
- [ ] The player can unlock exactly one valid second class for that base class
- [ ] The player can unlock exactly one valid specialization for the chosen second class
- [ ] Talent points can be invested in the correct active talent trees
- [ ] Progression choices remain tied to the selected class line
