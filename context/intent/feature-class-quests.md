# Feature: Class Unlock Quests

## What
Class unlock quests gate second classes and specializations behind dedicated quest lines. The player chooses a route, completes staged objectives, and earns themed rewards tied to the chosen path.

## Why
This feature makes class evolution feel earned instead of automatic. It reinforces class fantasy, gives structure to milestone progression, and provides memorable progression checkpoints.

## Acceptance Criteria
- [ ] A valid unlock quest becomes available when class progression conditions are met
- [ ] The player can choose one valid path for the active unlock stage
- [ ] Quest progress updates across combat and collection milestones
- [ ] Stage rewards follow the chosen class route
- [ ] Completing the quest unlocks the intended class advancement and blocks incompatible paths
