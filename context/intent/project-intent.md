# Project Intent: ANYTHING RPG

## What
ANYTHING RPG is a single-player text RPG played through a terminal interface. Players create a character, fight through dungeon runs, manage equipment and resources, unlock class evolutions, complete quests, and progress both combat and profession systems.

## Why
The project exists to provide a modular RPG that can grow through content additions instead of constant engine rewrites. The codebase and data layout suggest a strong focus on expanding classes, monsters, quests, items, and progression through editable game data.

## Current State
The project already includes a playable CLI loop with character creation, save/load, combat, dungeon progression, class evolution, talent trees, class quests, inventory and equipment, procedural loot, shops, crafting, gathering, procedural quest generation, and achievement tracking.

## Current Features
- Dungeon runs with room progression and boss cycles
- Combat with skills, items, casting, and status effects
- Three-stage class progression with talents
- Class unlock quests for second classes and specializations
- Inventory, equipment, rarity-based loot, shops, and economy
- Gathering professions and crafting recipes
- Procedural quest board generation
- Achievement tracking and lifetime statistics
